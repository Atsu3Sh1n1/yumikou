配管溶接DB見積金額計算機 使用説明書

概要

このツールは、配管施工における溶接工数（DB値）と見積金額を自動で算出する見積支援ツールです。 配管サイズごとの施工要素（ストレート・エルボ・チーズ/フランジ）を入力し、補正係数や単価に基づいて見積を行えます。

基本機能

  • サイズ選択： 呼び径25A〜200A
  • 部材数量入力： ストレート、エルボ、チーズ/FLの数量
  • DB値換算： 各部材のDB値を配管サイズごとに設定
  • スケジュール補正： SGP / SCH40 / SCH80 による倍率補正
  • 材質補正： ステンレスを選択するとさらに1.5倍補正（金額のみ）
  • 金額自動計算： 合計DBと金額をリアルタイムに表示

操作方法

  1. 配管サイズを選択（例：50A）
  2. 各部材の個数（ストレート・エルボ・チーズ）を入力
  3. 単価（円/DB）を設定（初期値：2000円）
  4. SCH補正ボタンを選択（SGP/SCH40/SCH80）
  5. ステンレス使用時は「ステンレス」ボタンを押下
  6. 合計DBと金額が下部に表示

計算ロジック

  • 各部材のDB値はサイズごとに定義（例：50Aのストレート＝2DB）
  • ストレート 1DB + エルボ 2DB + チーズ・フランジ 3DB
  • 合計DB × スケジュール倍率 × 材質倍率 = 補正後DB
  • 見積金額 = 補正後DB × 単価（円/DB）

用途・対象

  • 配管施工業者による積算見積
  • 監督者・設計者による工数評価
  • 現場でのDB計算と説明資料

注意事項

  • DB値は目安値であり、実際の施工条件により変動します。
  • フランジ溶接や現場状況による追加工は別途考慮してください。
