プラント配管見積もりマニュアル

ステップ1：前提情報の整理

  • 配管図の有無： P&ID、ISO配管図
  • 流体の種類： 蒸気、水、薬品など
  • 使用材質： SGP、STPG、SUS304 等
  • スケジュール： Sch.10 / 40 / 80
  • 圧力・温度条件： 耐圧材料が必要か判断
  • 施工条件： 屋外・屋内、高所作業の有無

ステップ2：数量の拾い出し

  1. 配管延長： 口径別にメートル単位で算出
  2. 継手の種類と数： エルボ、ティー、ユニオンなど
  3. バルブ類： 呼び径・種類ごとに分類
  4. 溶接点数： 口径・材質・スケジュールに応じて係数設定
  5. 支持金具： サポート点数と取付方法
  6. 非破壊検査範囲： PT/RT範囲を確認

ステップ3：単価・工数設定と見積集計

  • 材料費： 鋼管・継手・バルブなどの単価を反映
  • 溶接DB数設定： 材質・スケジュールごとに倍率
  • 工数換算： DB数 × 単位作業時間（人分）
  • 現場施工費： 高所・屋外は係数で加算
  • 管理・利益： 安全費・間接費・利益を15〜30％加算
