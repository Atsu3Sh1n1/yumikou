1. ビジネスメールの即時生成

定型メールを瞬時に作成する3ステップ手法

▽ 基本手順

1. 要件を箇条書きで整理

  • 件名のキーワード（例：発注依頼・納期確認・見積依頼）
  • 必須要素（品名・数量・希望納期・金額など）
  • 求めるトーン（丁寧・簡潔・確認重視など）

2. ChatGPTに入力する基本フォーマット

【件名】[ここに件名のキーワード]
【要件】
- 宛先：[取引先名/部署名]
- 目的：[用件を簡潔に]
- 具体的なデータ：[品名・数量・希望納期など]
- 特記事項：[納品先・支払い条件など]
- 文調：[丁寧/簡潔/依頼口調など]
- 文字数目安：[○字程度]

3. 出力結果のブラッシュアップ

  • 業界用語の確認（例：「ロット」「納品」「御見積」など）
  • 数量・納期・金額の整合性確認
  • 「明確さ」や「読みやすさ」重視で微調整

▽ 実践例：見積依頼メール

件名：アルミ部品加工の御見積依頼
【要件】
- 宛先：有限会社弓工 営業部 ご担当者様
- 発信者：株式会社コウセイ 技術購買課 山田
- 対象品目：アルミブラケット（型番：BR-550） 加工および表面処理含む
- 数量：ロット100個／200個の2パターンで希望
- 希望納期：2025年6月15日までに納品可能か
- 特記事項：図面（PDF）添付予定・納期優先・支払い条件は従来通り
- 文調：丁寧で要点を簡潔にまとめた依頼文
- 文字数：400字程度
1. テンプレート化

「このフォーマットでテンプレートを作成」と依頼すれば、よく使うメール型をストック可能

2. バリエーション生成

「3パターン作成（簡潔版・詳細版・英語版）」と指定すれば比較検討可能

3. 過去メールを学習

既存のメールを読み込ませ「この文体で新規作成」と依頼すれば自社フォーマットに準拠

▽ 注意点

  • 機密情報は含めない（実際の金額・個人情報など）
  • 生成後は必ず内容を確認（AIは事実確認ができない）
  • 特に謝罪文では責任所在が曖昧にならないよう注意

2. Excel関数の自然言語変換

関数が苦手でも日本語で質問するだけで解決。

以下の条件を満たすExcel関数を教えてください： - A列にテストの点数(0-100点)がある - B列に「合格」「不合格」を表示 - 合格基準：80点以上は「合格」、それ未満は「不合格」 - さらにC列にAランク/Bランク/Cランクを表示 - 90点以上：Aランク - 70-89点：Bランク - 69点以下：Cランク
「この関数をGoogleスプレッドシート用に変換して」と追加すれば、他のプラットフォーム向けにも対応可能。

3. 議事録の自動生成＆要約

雑多なメモから構造化された議事録を作成。

以下の会議メモを正式な議事録形式に整理してください： 【プロジェクト進捗会議 5/15】 - 参加者：山田、佐藤、鈴木 - 前回からの進捗：デザイン案完成、クライアントからOK - 課題：開発リソース不足、納期厳守が困難 - 決定事項：パートナー企業に外注、予算+15% - 次回MTG：5/22 14:00 出力形式： 1. 会議概要 2. 決定事項（★マーク） 3. アクションアイテム（担当者と期限） 4. 次回予定
「英語版も作成して」と依頼すれば多言語対応可能。音声文字起こしデータの整理にも最適。

4. マクロ/VBAコード生成

プログラミング知識不要で業務自動化。

以下の条件でExcelマクロを作成してください： 1. 「入力」シートのA列（名前）とB列（売上）データを処理 2. 売上が100万円以上の顧客を「優良顧客」シートに転記 3. 転記項目：名前、売上、日付（今日の日付自動入力） 4. 転記時に元データにチェックマークを付ける 5. ボタンクリックで実行できるようにする 6. 重複転記防止機能付き
「エラー処理を追加して」と指定すれば堅牢なコードに。PythonやGoogle Apps Scriptへの変換も可能。

5. 市場分析レポート作成

生データから洞察のある分析レポートを自動生成。

以下の売上データを分析し、A4サイズ1枚のビジネスレポート形式でまとめてください： 【2024年1-3月期 地域別売上】 - 関東：¥12,450,000（前年比+8%） - 関西：¥8,760,000（前年比-2%） - 中部：¥6,320,000（前年比+15%） - 九州：¥3,890,000（前年比+5%） 含める内容： 1. 総括（最大の成長地域と課題地域） 2. グラフ（棒グラフと円グラフのテキスト表現） 3. 次四半期への提言 4. 注目すべき数値3選
「PowerPoint用のスライド形式で」と指定すればプレゼン資料に。特定業界の専門用語も反映可能。

6. 社内連絡文の自動生成

定型連絡から緊急連絡まで、状況に合わせた文面を即時作成。

以下の条件で社内連絡文を作成してください： 【件名】社内ネットワークメンテナンスのご連絡 【本文要件】 - 日時：5月20日(月) 22:00～24:00 - 影響範囲：全社ネットワークが10分程度切断 - 対応：作業時間中のシステム利用を控える - 緊急連絡先：情報システム部 内線1234 - トーン：丁寧だが簡潔な事務連絡 - 追加情報：メンテナンス内容（サーバー更新） - 注意喚起：作業前にデータ保存を推奨 - 文字数：400字程度
「より簡潔に」「部門別にカスタマイズ」などの調整が可能。英語版やPDF形式での出力も依頼可。過去の類似連絡から文体を統一することもできます。

7. プレゼン資料の骨子作成

アイデアからすぐにプレゼン構成が完成。

「AIを活用した事務作業の効率化」というテーマで、10分間のプレゼン資料の構成を作成してください。 含める内容： 1. 現在の事務作業の課題（統計データ引用） 2. AI活用の具体的なメリット（3事例） 3. 導入ステップ（3段階） 4. 想定される懸念とその解決策 5. 成功事例（架空可） 各スライドのタイトルと、話す内容の要点を箇条書きで。
「各スライドの注釈として話すべきポイントを追加」でより詳細な資料に。PowerPointのデザイン提案も可能。

8. カスタマーサポート対応

クレーム対応からFAQ作成まで自動化。

以下の顧客クレームに対する適切な返信文を作成してください： 【顧客メール内容】 「先週注文した商品（注文番号#12345）がまだ届きません。 約束の納期は3営業日以内でしたが、既に5営業日経過しています。 至急状況を確認してほしい。」 返信条件： - 謝罪と原因説明（物流遅延） - 現在の状況（追跡番号付きで発送済み） - 補償案（次回10%割引クーポン） - 再発防止策 - 丁寧で誠実なトーン
「英語版も作成」「より簡潔なバージョン」など多様なバリエーションを生成可能。過去の対応例から学習させることも可能。

9. チェックリスト自動作成

作業の抜け漏れを防ぐ最適なリストを生成。

「新入社員の初日オリエンテーション」のチェックリストを作成してください。 項目： - 事前準備（1週間前〜前日） - 当日の流れ（午前・午後別） - 必要な書類と備品 - 導入すべき社内システム - 1週間後のフォロー項目 形式： - 表形式（項目、担当者、期限、備考） - 優先順位付き（★マーク）
「Googleスプレッドシート形式で」「完了欄付き」など用途に合わせてカスタマイズ可能。既存リストの改善提案も可。

10. データ分析＆可視化提案

生データから洞察と適切な可視化方法を提案。

以下の売上データを分析し、最も効果的な可視化方法を提案してください： 【2023年 月別商品別売上（単位：万円）】 月,商品A,商品B,商品C 1月,120,85,65 2月,135,78,72 ... 12月,210,95,110 求める出力： 1. 主要なトレンドの指摘（3点） 2. 推奨グラフタイプとその理由 3. 注目すべき異常値 4. 次年度への提言（2点）
「Pythonコード付きで」「Tableau用に」と指定すれば分析コードも自動生成。予測モデルの構築も可能。

プロンプト作成の3大原則

  1. 具体的である - 「良い」ではなく「80点以上」と数値化
  2. 出力形式を指定 - 箇条書き、表形式、文字数など
  3. 文脈を提供 - 対象読者や使用目的を明記

これらの原則を守れば、ChatGPTの出力品質が劇的に向上します。
